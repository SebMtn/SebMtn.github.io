Get Involved

This project is critically important but very ambitious. We need all the help we can get.

Here are some ways you can help:

  • Express your support by giving a Star to the GitHub project

  • Join the discussion on our GitHub Discussions. Any comment helps to build the community.

  • Report any typo you find on this website.

  • Make suggestions for better ways to phrase some sentences on this website.

  • Translate some parts of this website to a language you are familiar with. Any amount of translation helps, you do not have to translate the entire website, or even an entire page.

  • Suggest what companies might be interested in trying out our model.

  • Do you have technical expertise that could be relevant? Please make suggestions. We are grateful for any tip.

  • Can you see flaws in this model? Areas of improvements? Please tell us. Any feedback welcome.
